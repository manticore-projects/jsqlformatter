************
Installation
************

Git
===================
.. code:: Bash
    
   git clone https://github.com/manticore-projects/jsqlformatter.git
   cd jsqlformatter
   
.. tabs::

   .. tab:: Maven 

      .. code:: Bash
      
        mvn install

   .. tab:: Ant

      .. code:: Bash

        ant jar


Maven Repo
===================

.. code-block:: xml
   :substitutions:
     
    <dependency>
        <groupId>com.manticore-projects.jsqlformatter</groupId>
        <artifactId>jsqlformatter</artifactId>
        <version>|VERSION|</version>
    </dependency>
